分享踩过的坑

我的这个博客是利用免费开源的 GitHub Pages 搭建的，需要先注册一个账号。

GitHub

我用 GitHub 也不是特别久。GitHub 是通过Git进行版本控制的软件源代码托管服务。是一个成熟的开源社区。现已被微软收购。

Github Pages

Github Pages可以被认为是用户编写的、托管在GitHub上的静态网页，合理搭配Hexo也可以做一个炫酷的个人网站。

普通建站需要一台服务器，支付一定的费用，自行搭建运行环境。不仅耗时，而且费钱，做不好还打击积极性和自尊心。比如我当初利用 WordPress （当然，这也是一种非常好的建站方式）搭建网站的时候，是非常折腾人的，什么PHP、MySQL、Nginx、数据库等等…… 看着就头大，还好现在很多公司都提供了可视化面板，把几个小时的建站时间缩短为几十分钟，甚至更低，譬如，宝塔面板。

言归正传……

注册GitHub账号

注册页面：https://github.com/join

GitHub上新建项目

注册好之后，右上角 新建代码仓库

接下来填写详情，仓库名字一定要是 xxx.github.io 这样格式的，xxx是你自己的名字，比如我的就是ithou.github.io，这是以后访问博客的域名

现在就可以通过你的域名 http://xxx.github.io在浏览器访问了，可以看到除了标题什么都没有。（但还是成功了一大半啦~ ）
