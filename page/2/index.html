Python 学习笔记 零 ——代码风格

发表于 2018-11-07 | 分类于 Code
字数统计: 150

  1. 风格如影随形，因人而异，但是有一个统一的PEP 8风格

  2. 变量命明易懂是一种习惯，默默遵守，方便自己，方便他人

  3. 尽量少使用abc…一类的无意义变量名

  4. PEP 8 建议每级缩进都使用4个空格（编辑器中为一个tab键）

  5. 绝对不要混用 tab 和空格，使用任何编辑器写Python，请把一个 tab展开为 4 个空格

  6. 行长：最多80字符，最长最长不超过编辑器右边的竖线——不可逾越的红线
